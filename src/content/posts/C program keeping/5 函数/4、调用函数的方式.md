### 4.1、直接调用函数

**调用的方式为：**

    函数名();

- 如果调用的函数 在定义时有形参，那么在调用的时候就应该传递参数
- 调用时，实参的个数和先后顺序应该和定义函数中要求的一致

### 4.2、嵌套调用函数

    C 语言中函数之间是平行的 ，因此不能嵌套定义，但是函数之间可以互相 调用(嵌套调用)，习惯上把调用者称作`主调函数`；

也就是一个函数中可以调用另外一个函数；

![[_attachments/Pasted image 20240723220339.png]]

![[_attachments/Pasted image 20240723220459.png]]
**示例：**

```c
#include "stdio.h"

int get_data(int num1, int num2, int num3) {
    int result;
    result = num1 + num2 + num3;
    return result;
}

int average(int num1, int num2, int num3) {
    int data;
    data = get_data(num1, num2, num3);
    return data / 3;
}

int main() {

    int data;
    data = average(11, 22, 33);
    printf("%d\n", data);

    return 0;
}
```

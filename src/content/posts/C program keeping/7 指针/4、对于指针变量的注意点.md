### 4.1、定义指针变量时必须要有`*`

如果在一行语句中，定义多一个指针变量，那么每个变量前都要有`*`

```c
#include "stdio.h"

int main() {

    char *p1, p2;

    printf("p1变量的内存大小:%d, p2变量的内存大小:%d\n", sizeof(p1), sizeof(p2));

    return 0;
}
```

### 4.2、指针变量赋值时不加`*`
